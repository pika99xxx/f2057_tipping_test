# F2057 防倾倒测试预判工具

这是一个基于 Streamlit 开发的柜类产品防倾倒测试预判工具，用于在产品设计阶段快速评估柜类产品是否存在倾倒风险，并辅助判断是否可能符合 ASTM F2057-23 的相关测试要求。

> 注意：本工具仅用于设计阶段的工程预判，不能替代正式实验室测试。

在线使用 https://f2057tippingtest-pika99xxx.streamlit.app/

## 功能介绍

本工具主要用于柜类产品，例如：

- 五斗柜
- 衣柜
- 床头柜
- 梳妆台
- 储物柜
- 带抽屉或柜门的柜类产品

核心功能包括：

- 判断产品是否属于 ASTM F2057-23 测试范围
- 自动估算整柜重量
- 计算顶板、底板、侧板、背板、抽屉、门板等板件重量
- 支持 PB 板、MDF 板、吸塑门板、玻璃门等不同材料密度
- 计算封闭存储体积
- 判断模拟衣服荷重是否生效
- 计算抽屉 OSL 60% 拉出状态下的倾覆风险
- 计算柜门打开角度下的倾覆风险
- 支持玻璃门、板式门、板式+玻璃混合门
- 支持拉手位置自动计算拉手高度
- 模拟 44N 水平动态拉力测试
- 模拟 60lb 儿童荷重逐一测试
- 考虑后脚测试块导致的重心前移影响
- 计算所有抽屉和柜门同时打开时的风险
- 检查内锁装置、防倾倒五金、警告标贴等合规项
- 自动生成测试预判报告并支持下载

## 使用场景

本工具适合用于：

- 家具产品开发前期风险筛查
- 柜类产品结构设计评估
- 工厂打样前的稳定性预判
- 产品专员、结构工程师、品质人员协同评审
